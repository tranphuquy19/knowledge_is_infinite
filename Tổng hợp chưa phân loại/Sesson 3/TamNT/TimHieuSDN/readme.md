# Thư mục chứa các ghi chép tìm hiểu về công nghệ Mạng định nghĩa bằng phần mềm (Software Defined Networking - SND)

Cấu trúc thư mục

[`docs`](./docs): thư mục chứa các ghi chép tìm hiểu được.
